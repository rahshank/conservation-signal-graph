## Live-source gate
The NPS webcam API is implemented as the first candidate. After adding `NPS_API_KEY`, Yosemite (`NPS_PARK_CODE=yose`) returned a live-camera source for Yosemite Falls. Yellowstone, Grand Canyon, Acadia, and Glacier did not return an image URL in the first probe.

The dashboard now includes an `Ingest NPS source` action. With `CSG_FORCE_FIXTURE=1`, that action ingests the real NPS source metadata and uses fixture extraction. With `CSG_FORCE_FIXTURE=0`, Groq extraction has been proven against Yosemite Falls and Bartlett Cove Lagoon.

Local API proof:

- `POST /api/events/ingest/nps` returned a dashboard state with `liveEvents: 1`.
- Source: Yosemite Falls, Yosemite National Park.
- Extraction mode: `fixture`.
- Graph mode: `memory`.

## Groq Extraction Proof

Yosemite Falls proof:

- Source: Yosemite Falls, Yosemite National Park.
- Source mode: `live_camera`.
- Model: `meta-llama/llama-4-scout-17b-16e-instruct`.
- Prompt version: `csg-v0.1`.
- Latency: 1235 ms.
- Validation status: `valid`.
- Summary: no wildlife observed in the image.
- Learning: Yosemite Falls proves the source and Groq path, but it is weak as a wildlife signal source.

Bartlett Cove Lagoon proof:

- Source: Bartlett Cove Lagoon and Fairweather Range, Glacier Bay National Park & Preserve.
- Source mode: `live_camera`.
- Model: `meta-llama/llama-4-scout-17b-16e-instruct`.
- Prompt version: `csg-v0.1`.
- Latency: 1558 ms.
- Validation status: `valid`.
- Summary: no wildlife observed in the current image.
- Open questions generated: representative habitat view, visible pollution or human impact, timestamp and location accuracy.
- Learning: Bartlett Cove is a better source candidate than Yosemite Falls, but one current frame still may not contain wildlife. The product needs repeated sampling, a short observation window, or a curated evidence path for an interesting frame.

Bird nest known-context proof:

- Peregrine Falcon Webcam:
  - Expected context: peregrine falcon / nest camera.
  - Original image URL: `https://www.nps.gov/common/uploads/cropped_image/FCD71ADD-A61D-57A9-C8A70C735880F813.jpg`.
  - Original image: 8640 x 5760, 11,054,549 bytes.
  - Submitted image after normalization: 1600 x 1067, 133,007 bytes.
  - Model: `meta-llama/llama-4-scout-17b-16e-instruct`.
  - Prompt version: `csg-v0.1`.
  - Latency: 1316 ms.
  - Validation status: `valid`.
  - Summary: Peregrine Falcon in flight.
  - Species candidate: Peregrine Falcon, confidence 0.99.
- Bald Eagle Webcam, Sauces Canyon Nest:
  - Expected context: bald eagle / nest camera.
  - Original image URL: `https://www.nps.gov/common/uploads/cropped_image/22CD2AFA-B573-600F-C221D0D218D39CDD.jpg`.
  - Original image: 4267 x 2400, 4,341,288 bytes.
  - Submitted image after normalization: 1600 x 900, 267,439 bytes.
  - Model: `meta-llama/llama-4-scout-17b-16e-instruct`.
  - Prompt version: `csg-v0.1`.
  - Latency: 1352 ms.
  - Validation status: `valid`.
  - Summary: Bald Eagle with chicks in nest.
  - Species candidates: Bald Eagle, confidence 0.95; Eagle chick, confidence 0.8.
  - Actions suggested: Nest maintenance; chick care.
- Learning: nest cams are better known-context tests than scenic webcams. Product image normalization now prevents oversized NPS source frames from failing Groq vision calls.

## Run-detail UI

The dashboard now shows the latest observation's model run details in the working surface:

- model
- prompt version
- latency
- validation status
- confidence
- graph mode
- extraction mode
- source URL or fixture asset
- frame-processing metadata when a model run includes normalized-frame data

The Playwright coverage includes the fixture state and a mocked Groq-style model run with original dimensions, original bytes, submitted dimensions, submitted bytes, resized status, and normalization reason. The browser matrix was also rerun after the UI change.

Code review found two issues: source URL display shortened traceable URLs, and Playwright did not assert enough visible telemetry values. The fixes now render the exact source URL as a link and assert model, latency, validation value, confidence, graph mode, extraction mode, source URL, and frame-normalization values.
